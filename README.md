# sequelize-view-builder

Generador programático de vistas SQL complejas usando modelos Sequelize + TypeScript.

## Instalación

```bash
npm install sequelize-view-builder
```

## Ejemplo básico

```ts
import { ViewBuilder } from 'sequelize-view-builder';
```

## Comandos

```bash
npm run build           # Compila la librería
npm run example         # Ejecuta ejemplo básico
```
